## 实战题：链表随机节点

题意：给定一条长度未知的单链表，随机返回一个节点值；每个节点被返回的概率必须相同。

映射：第 `i` 个访问到的节点，以 `1/i` 的概率替换当前答案。

## 核心代码

```java
int selected = nums[0];
Random random = new Random();

for (int i = 1; i < nums.length; i++) {
    if (random.nextInt(i + 1) == 0) {
        selected = nums[i];
    }
}
```

## 准确性要点

- 第 `i + 1` 个元素到来时，用 `random.nextInt(i + 1) == 0` 表示 `1/(i+1)`。
- 第 `k` 个元素最终留下概率是：

```text
1/k * k/(k+1) * (k+1)/(k+2) * ... * (n-1)/n = 1/n
```

- 选 1 个样本：时间复杂度 O(n)，空间复杂度 O(1)。
- 选 k 个样本：时间复杂度 O(n)，空间复杂度 O(k)。
